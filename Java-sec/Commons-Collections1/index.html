Commons-Collections是什么

Commons-Collections是Apache Commons Collection的简称，是Apache软件基金会开发的一个开源Java库，提供了一系列Java集合类的实现。

这个库主要提供了如下几类集合类的实现：

  • 序列：这个包提供了一些常见的序列实现，比如堆栈、队列和双端队列。
  • 迭代器：这个包提供了一些常见的迭代器实现，比如过滤器迭代器和反向迭代器。
  • 图形：这个包提供了一些常见的图形实现，比如树和图。
  • 视图：这个包提供了一些常见的视图实现，比如只读视图和子视图。

Apache Commons Collection库还提供了一些其他的功能，比如对Java集合类的扩展、工具类和接口。这个库可以帮助Java开发人员更方便地使用Java集合类，并提供了一些常见的集合类实现，使开发人员可以更快速地开发应用。

  • var2：调用的方法。
