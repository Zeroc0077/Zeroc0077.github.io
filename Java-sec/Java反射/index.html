Java反射

慢慢入门Java-SEC吧

基础概念

java反射机制是反序列化的基础知识，这里介绍一些java反序列化中的一些常见概念：

  • source：入口
  • gadget chain：调用链
  • sink：目标方法
  • RMI：Remote Method Invocation远程方法调用

利用反序列化漏洞就是在已知source和sink的情况下，如何走通整个流程的问题。

在CTF的题目中，我们一般都是希望通过反序列化进行RCE等操作。

常见的source包括：

  • java的原生反序列化：通过ObjectInputStream.readObject()来处理二进制格式的序列化内容得到反序列化后的java对象。
  • Fastjson、Xstream等第三方库：处理json、xml等格式的内容，得到java对象。

sink包括：

  • Runtime.exec()，能够直接在目标环境中进行命令执行。
  • Method.Invoke()，通过反射执行。
  • RMI/JNDI/JRMP，通过远程引用对象，间接命令执行。
